Connect to your repo

Publishing writes data/videos.json straight into your GitHub repo. Pages rebuilds and the site updates — usually within a minute.

Make a fine-grained token at github.com → Settings → Developer settings → Tokens. Give it access to this one repo only, with Contents: Read and write. Nothing else.
The token lives in this browser only. It is never written into the repo, so it is safe to host admin.html on the public site — anyone can open the page, but without their own token they cannot change a thing.
